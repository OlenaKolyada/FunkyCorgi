Some of my projects

Grem – Video Game News & Reviews Portal

GREM is a desktop-first web platform fully developed by me from scratch during my internship as part of the Développeur Web et Web Mobile program at AFPA. The project includes both the frontend and a custom REST API on the backend. Its goal is to provide users with news and original game reviews written by an expert, offering detailed and independent analysis.

The project is currently under development.

Technologies used:

  • Frontend: HTML5, CSS3, JavaScript, React, Next.js
  • Backend: PHP, Symfony, MySQL
  • Deploy: AWS, WampServer, Docker
  • Version control: Git, GitHub
  • Testing: Postman
  • Wireframes / UML: Figma, Draw.io
  • IDE: PHPStorm, VS Code

Authentication System – POC

This login and password management system was developed during my Développeur Web et Web Mobile training. It allows users to register, confirm their email, log in, and reset their password if needed. The project uses object-oriented programming, an MVC architecture (controllers, models, views, services), secure token-based password recovery with expiration, and email sending via PHPMailer.

The mobile version is currently under development.

Technologies used:

  • Frontend: HTML5, CSS3, JavaScript
  • Backend: PHP, Symfony, MySQL
  • Deploy: FileZilla
  • Testing: Postman
  • Version control: Git, GitHub
  • IDE: VS Code
