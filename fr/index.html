Quelques-uns de mes projets

Grem – Portail d'actualités et de critiques de jeux vidéo

GREM est une plateforme web prioritairement conçue pour le bureau, entièrement développée par moi-même de A à Z lors de mon stage dans le cadre du programme Développeur Web et Web Mobile à l’AFPA. Le projet comprend à la fois le frontend et une API REST personnalisée côté backend. Son objectif est de fournir aux utilisateurs des actualités et des critiques de jeux originales rédigées par un expert, offrant une analyse détaillée et indépendante.

Le projet est actuellement en cours de développement.

Technologies utilisées :

  • Frontend : HTML5, CSS3, JavaScript, React, Next.js
  • Backend : PHP, Symfony, MySQL
  • Déploiement : AWS, WampServer, Docker
  • Gestion de version : Git, GitHub
  • Tests : Postman
  • Maquettes / UML : Figma, Draw.io
  • IDE : PHPStorm, VS Code

MLBA – Site vitrine pour une académie de ballet

MLBA est un site vitrine créé bénévolement pour une académie de ballet. Son but est de présenter les activités et services de l’académie, ainsi que de fournir des formulaires de contact et d’inscription. Le thème WordPress a été développé sur mesure par moi-même à partir de zéro, en utilisant le thème de démarrage Underscores (_s) comme base.

Le projet est actuellement en cours de développement.

Technologies utilisées :

  • Déploiement : FileZilla

Système d’authentification – POC

Ce système de gestion des connexions et mots de passe, développé durant ma formation Développeur Web et Web Mobile, permet aux utilisateurs de s’inscrire, confirmer leur email, se connecter et réinitialiser leur mot de passe grâce à une architecture MVC orientée objet, une récupération sécurisée par token expirant, et l’envoi d’emails via PHPMailer.

La version mobile est actuellement en cours de développement.

Technologies utilisées :

  • Frontend : HTML5, CSS3, JavaScript
  • Backend : PHP, Symfony, MySQL
  • Déploiement : FileZilla
  • Tests : Postman
  • Gestion de version : Git, GitHub
  • IDE : VS Code
